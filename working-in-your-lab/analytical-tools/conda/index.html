# Additional resources

  • Install R with conda
  • Update to the latest R version
  • Please note that this does work with some, but not all, bioconductor-packages. Please send us a note if you need these. List of available r-packages here
# install
conda install r-essentials
conda update r-essentials
conda install -c conda-forge r-base

# install cran packages within r
install.packages("png", "~/anaconda3/lib/R/library")
