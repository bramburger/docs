# Conda, Anaconda, Miniconda and Bioconda

Conda - Anaconda - Miniconda - Bioconda

# Installation

# Install Conda

Follow the Installing on Linux instructions to install Miniconda.

To get more out of conda we recommend reading Conda - Managing environments guide.

# Set up Bioconda and other channels

Follow the Set up channels guide from Bioconda.

conda config --add channels defaults
conda config --add channels bioconda
conda config --add channels anaconda
conda config --add channels conda-forge

Depending on your aims you might also consider adding R channel.

conda config --add channels r

# Install Your Tools

Have a field day installing your favourite tools

conda install plink2

# Do Cool Science

plink2 <input>

# Additional resources

  • Install R with conda
  • Update to the latest R version
  • Please note that this does work with some, but not all, bioconductor-packages. Please send us a note if you need these. List of available r-packages here
# install
conda install r-essentials
conda update r-essentials
conda install -c conda-forge r-base

# install cran packages within r
install.packages("png", "~/anaconda3/lib/R/library")
