# Rsync

RSync (opens new window) is a fast and extraordinarily versatile file copying tool. It can copy locally, to/from another host over any remote shell, or to/from a remote rsync daemon.

# Install on Linux (Ubuntu)

sudo apt install rsync

# Install on Mac OS X

REQUIREMENT

Install Homebrew package manager for macOS: brew (opens new window)

brew install rsync

# Usage

RSync (opens new window) provides many parameters described in their documentation. We provide only simple example to start with when transferring files from your computer to remote machine (server):

rsync -avuz path/to/directory username@example.com:/target/path/
